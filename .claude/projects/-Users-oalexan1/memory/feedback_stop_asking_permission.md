Do not ask "want me to edit X?" or "should I add this?" when editing CLAUDE.md, MEMORY.md, .bashrc, .zshrc, settings files, or similar config files. Just do it.

**Why:** The user has explicitly granted permission for these edits. Asking is redundant and annoying.

**How to apply:** When the user says to remember something, add a rule, or modify config — edit the file directly without asking for confirmation.
