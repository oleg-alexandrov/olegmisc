XEmacs tips

I have found XEmacs to be the best text editor on UNIX. It has very powerful editing features, and virtually everything in XEmacs is customizable. The drawbacks are that the default XEmacs behavior is very unusual to a new user, and it takes a lot of time and knowledge to customize it. In these pages I hope to give some tips about how to make XEmacs behave more predictably, while showing that it has some features which make it a great choice for an editor.

I believe everything here should work on XEmacs 24 or later. Some things will not work on Gnu Emacs.
