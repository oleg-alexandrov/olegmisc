Customizing the XEmacs Toolbar

I modified the XEmacs toolbar by changing the icons (I am using the KDE Crystal theme icons) and by adding the buttons which perform the redo and search functions, while removing some functions I never used, like news or debug. Here's the result.



The actions the buttons perform are (from left to right)
|open|save|print|cut|copy|paste|undo|redo|spell|search|replace|mail|

The emacs lisp code needed to install this toolbar is here. The icons can be found here. One can install these for instance in the following way: save the file at the link above as my-toolbar.el in the directory .xemacs. Put the icons in a directory called icons in .xemacs. Insert the following line in init.el: 

(load-file "~/.xemacs/my-toolbar.el"); load the toolbar
